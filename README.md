# LoLAPI-Project
A project taken on to sharpen skills on React.js &amp; Node.js

### Front-End 
PORT 3000
React.js 

### Back-End
PORT 3001
Node.js Version 20.9.0 LTS



### How to setup

After installing correct node version: 

- npm install


### To Run Front-End
- cd client
- npm start

### To Run Back-End
- cd "root directory"
- npm start
